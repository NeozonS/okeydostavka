OkeyDostavka Parser

Парсер для сбора всех товаров с сайта okeydostavka

Укажите ID Магазина или координаты доставки что бы выбрать ближайший магазин в радиусе доставки. Так же укажите прокси.


ПРОКСИ
```
PROXY=http://username:password@proxy.example.com:8080
Или без авторизации: PROXY=http://192.168.1.100:8080
```
----
Выбор магазина:
Вариант 1: Координаты доставки
```
LATITUDE=59.9386
LONGITUDE=30.3141
```
Вариант 2: Прямой выбор магазина (если не заданы координаты)
```
ID_STORE=13151  # Москва по умолчанию
```
----
Использование
Пример запуска с координатами:
```
PROXY=http://username:password@proxy.example.com:8080 \
LATITUDE=59.9386 \
LONGITUDE=30.3141 \
./okey_parser
```
----
Пример запуска с выбором магазина по ID:
```
PROXY=http://192.168.1.100:8080 \
ID_STORE=13151 \
./okey_parser
```
